Aplicación web que permite buscar información de cualquier país del mundo usando la API de RestCountries.

## 🚀 ¿Qué hace?
 
- Busca cualquier país por nombre
- Muestra la bandera, capital, población, región y subregión
- Muestra un mensaje de error si el país no existe
- Diseño responsive — funciona en móvil y desktop
 
---
 
## 🛠️ Tecnologías usadas
 
- **HTML5** — estructura semántica
- **CSS3** — variables CSS, Flexbox, animaciones, media queries
- **JavaScript** — manipulación del DOM, Fetch API, async/await
- **RestCountries API** — fuente de datos de los países

## ▶️ ¿Cómo usarlo?
 
1. Clona el repositorio:
```bash
git clone https://github.com/tu-usuario/buscador-paises.git
```
 
2. Abre la carpeta del proyecto:
```bash
cd buscador-paises
```
 
3. Abre el archivo `index.html` en tu navegador o usa Live Server en VS Code.
 
---
 
## 🌐 API utilizada
 
[RestCountries](https://restcountries.com/) — API gratuita que proporciona información de todos los países del mundo.
 
Endpoint usado:
```
https://restcountries.com/v3.1/name/{nombre-del-pais}
```
